## Flood Prediction

Проект посвящён разработке алгоритма прогнозирования уровня воды и риска паводка на основе исторических временных рядов наблюдений у мостов.

Решение включает анализ данных, предобработку временных рядов, построение признаков и обучение модели машинного обучения для оценки будущего уровня воды и потенциальной паводковой угрозы.
