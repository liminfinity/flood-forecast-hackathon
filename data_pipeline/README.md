# Data Pipeline for Flood Forecast

Этот модуль подготавливает единый датасет для модели прогнозирования паводков.

Что делает пайплайн:
1. Читает и очищает исходный датасет с уровнем воды.
2. Приводит временные метки к удобному формату.
3. Загружает исторические погодные данные из Open-Meteo по координатам мостов.
4. Объединяет уровень воды и погоду в один датасет.
5. Строит базовые признаки для модели:
   - lag features
   - rolling features
   - временные признаки
   - агрегаты осадков и температуры

## Структура

- `config/bridges.example.csv` — пример файла с координатами мостов
- `src/clean_base_dataset.py` — очистка исходного датасета
- `src/fetch_openmeteo.py` — загрузка погодных данных
- `src/merge_features.py` — объединение и генерация признаков
- `src/build_dataset.py` — запуск всего пайплайна одной командой

## Подготовка

Создай файл `config/bridges.csv` по примеру `config/bridges.example.csv`.

Пример:
